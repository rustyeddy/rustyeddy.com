Current Work

I currently work at Radian Laser Systems, LLC on various software componenets, I am responsible for software design, developement, deploy and support of machine software.

The software is a realtime, embedded system with many independent working parts that must be choreographed in real-time while ensuring the saftey of the people and equipment itself.

In the process of building this modern, but rather large “IoT” device, we are working through the challenges of managing and automating backend support systems and building intuitive Human Displays or GUI’s.

Sensor Station

Sensor Station is my pet project that gets a lot of my attention, when I can carve out time.

The project is an Automated Gardening Application that I started for two reasons:

  1. I wanted to find a practical way to grow some of my own food in my very small coastal southern california backyard.
  2. I wanted a 100% OpenSource piece of software (as it turns out pieces of software) that could use for my portfolio.
