Overview

MobileRobot is a Robotics project to build an affordable robot that can be controlled remotely as well as autonomous control (with driving algorithms). A primary goal for the robot is toting around a camera suitable for high definition shots and streaming video.

Operational Requirements

  1. The robot must be powered with out a cable such as with a battery
  2. The robot must to be able to run autonomous
  3. The robot must support a remote control
  4. The robot must be able to carry a camera
  5. The robot must be able to carry sensors
  6. The robot must be able to store

Mobile Vehical

The vehical must be affordable which is a subjective term for sure.
