WPE an Excellerated Raspberry Pi Kiosk

Raspberry Pi's have just become even cooler! Add an inexpensive touch panel and you can create insanely cool.

The WPE project (TODO: place reference) allows you to create a hardware accelerated web viewing_ experience with an embedded system and inexpenive display.

For example a Raspberry Pi with an 7inch tftp touch screen has my brain racing with ideas for applications.

About WPE

WPE creates a cool hardware accelerated kiosk, just point at a URL and the app displays the application takes over the screen.

It is awesome! But, it has one major drawback.!.
